# realestatekit

Turn a Zillow listing URL into a complete real-estate marketing kit:
landing page · 3-aspect Ken-Burns video · print PDFs (brochure / leave-behind / postcard) ·
4 social images · email blast · captions · talking points · post schedule ·
hosted Kit Dashboard URL she opens on her phone.

White-label per agent. Deployed to Netlify (per-listing tier) or her own custom
domain (monthly retainer tier).

This repo is the skill source — drop it under `~/.claude/skills/zillow-clone/` to use
it as a Claude Code skill, or run the Python renderers directly.

  social/
    fb-feed-1080.png             ← 1080×1080 Facebook feed
    ig-feed-1080.png             ← 1080×1080 Instagram feed
    ig-story-1080x1920.png       ← 1080×1920 Instagram Story
    reels-cover-1080x1920.png    ← 1080×1920 Reels cover with TAP TO TOUR CTA

    facebook-first-comment.txt   ← URL-only first comment (clean post pattern)

    email-blast.txt              ← Plain-text fallback

## Quick start

Requirements:
- Python 3.10+
- `pip install playwright qrcode[pil] pillow` then `python -m playwright install chromium`
- Windows (uses Cambria as the social-image font fallback) OR drop a real `Cormorant-SemiBold.ttf` into `assets/`

Run the full kit on a listing:

```bash
python kit-dashboard/render_dashboard.py \
  --listing path/to/listing.json \
  --agent kim-smith \
  --slug 148-pheasant-run-paducah-ky \
  --out ~/Downloads/148-pheasant-run-paducah-ky-kit
```

Outputs the complete kit at `~/Downloads/148-pheasant-run-paducah-ky-kit/`.

Open `index.html` in a browser, or deploy the whole directory to Netlify and you have
the agent's private dashboard URL.

## Tier-aware white-label

Each agent has a JSON brand profile in `agents/<slug>.json`:

```json
{
  "agent_name": "Kim Smith",
  "brokerage": "Smith & Co. Realty",
  "logo_path": "agents/kim-smith/logo.png",
  "headshot_path": "agents/kim-smith/headshot.jpg",
  "primary_color": "#2c5e3f",
  "accent_color": "#c9a449",
  "phone": "270-555-0123",
  "email": "kim@smithcorealty.com",
  "license_number": "KY-12345",
  "tier": "monthly",
  "custom_domain": "listings.smithcorealty.com",
  "footer_tagline": "Smith & Co. Realty · Paducah's trusted real estate"
}
```

Tier behavior:
- `per-listing` — dual-branded, small "Listing Kit by getoios.com" footer line
- `monthly` — full white-label, ZERO third-party branding, deploys to her custom domain

Brand profile flows everywhere automatically: print PDFs, social images, email HTML,
landing page CSS variable overrides, QR target URL, OG share preview titles.

See `agents/_example-template.json` for the full schema and `agents/_default.json`
for the per-listing tier defaults.

## Architecture

```
print/
  render_pdfs.py          ← Playwright HTML→PDF, generates QR
  brand_profile.py        ← Loader + tiny mustache template renderer
  brochure.html           ← 8-page brochure template
  one-sheet.html          ← Front+back leave-behind template
  postcard.html           ← 4×6 postcard template
  _listing_example.json   ← Smoke-test fixture

social/
  render_social_images.py ← Pillow + Cambria/Cormorant text overlays

email/
  render_email.py         ← Subject + preview + HTML + plain-text
  blast.html              ← Inline-styled email template

kit-dashboard/
  render_dashboard.py     ← MASTER ORCHESTRATOR — runs everything
  index.html              ← Kit Dashboard with copy/download buttons

agents/                   ← Per-agent brand profiles (white-label)
assets/                   ← Shared fonts (Cormorant) + music bed for video pipeline

SKILL.md                  ← Claude Code skill spec (triggers, inputs, full pipeline docs)
```

## Skill triggers (when used as a Claude Code skill)

- `clone this Zillow listing: <URL>`
- `/zillow-clone <URL>`
- `/zillow-clone <URL> with print kit`
- `/zillow-clone <URL> with full kit and host it`
- `/zillow-clone <URL> as kim-smith with full kit and host it`

The skill handles the upstream Zillow `__NEXT_DATA__` extraction, photo classification,
narrative generation, and video pipeline. The renderers in this repo handle every
downstream marketing asset.

## Pricing tiers (the why)

This kit exists to support a 3-tier real-estate offering:

| Tier | What | Notes |
|---|---|---|
| **Digital-only** | Landing page + 3-aspect video + FB caption + share preview | Per-listing pricing, dual-branded |
| **Listing Kit** | Everything above + print PDFs + social pack + email + photo pack + dashboard | Per-listing, dual-branded — the "binder package" |
| **Monthly Retainer** | Unlimited Listing Kits + full white-label + custom domain + her own listings.her-domain.com | Zero OIOS branding, agent's own URL |

Marginal cost per listing: ~$0–$0.50.
